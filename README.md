# Installation

## Setting up

### Debian

- Install dependencies for the installer script

```bash
sudo apt update && sudo apt upgrade && sudo apt install python3 wget -y
```

- Download the script

```bash
wget  "https://raw.githubusercontent.com/hirusha-adi/Near/main/setup.py"
```

- Run the script

```bash
python3 setup.py
```
